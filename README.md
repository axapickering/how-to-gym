[Next.js](https://nextjs.org/docs) app 
using [MUI Core](https://mui.com/material-ui/getting-started/) React component library

## Getting Started

Start the server:

```bash
npm run dev
``` 

Open [http://localhost:3000](http://localhost:3000) in browser
